# 버튼잡기게임(C# 코딩)
## 개요- C# 프로그래밍 학습
- 핵심기능: 마우스가 버튼 위로 올라갔을시 랜덤 좌표로 이동, 버튼 좌표 상단에 표시, 버튼 클릭시 축하메세지 출력
- - 화면구성: "나를 잡아봐"버튼, 술래잡기 타이틀
